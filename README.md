# tableau-data-analysis-dashboard
I created an interactive Tableau dashboard using the Kaggle Shopping Behavior dataset to analyze customer purchase patterns, product preferences, and seasonal trends. Multiple sheets were developed and combined into a Tableau Story to present insights in a structured and visual manner.



# Shopping Behaviour Data Analysis (Tableau)

## Project Overview
This project analyzes the Shopping Behavior dataset from Kaggle using Tableau to understand customer purchasing patterns and preferences.

## Objectives
- Analyze customer demographics and purchase behavior
- Identify trends in product categories and seasonal purchases
- Visualize patterns through interactive dashboards

## Tools Used
- Tableau
- Excel / CSV dataset
- Data Visualization
- Exploratory Data Analysis (EDA)

## Dashboard Features
- Customer demographic analysis
- Product category trends
- Seasonal purchase patterns
- Interactive filters for exploring insights

## Project Structure
- Tableau packaged workbook (.twbx)
- Dataset file
- Dashboard screenshots

## Key Outcome
The dashboard presents insights about how customer attributes influence purchasing behavior and helps visualize trends in shopping patterns.
